<h1> Business Intelligence • Data Visualization • Insights Estratégicos </h1>

Este repositório reúne projetos desenvolvidos em Power BI, com foco em análise de dados, visualização interativa e geração de insights para tomada de decisão.
Os dashboards apresentados exploram diferentes contextos de negócio, aplicando boas práticas de modelagem de dados, DAX, storytelling com dados e design analítico, sempre priorizando clareza, usabilidade e valor analítico.
Cada projeto contém a descrição do problema, as métricas analisadas e os principais aprendizados obtidos a partir dos dados.
